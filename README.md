# ChatRTC

#### Git clone

```
git clone https://github.com/andreyskorchenko/ChatRTC.git
```

#### Change directory

```
cd ChatRTC
```

#### Make .env file

```
echo "VITE_SOCKET_URL=http://localhost:3000" > ./frontend/.env
```

### Run in docker
> [!TIP]
> ```
> docker compose up -d
> ```

### Run local dev
> [!TIP]
> Frontend
> ```
> cd frontend
> ```
> ```
> npm ci
> ```
> ```
> npm run dev
> ```
> Backend
> ```
> cd backend
> ```
> ```
> npm ci
> ```
> ```
> npm run start:dev
> ```
